# DSA Repository

This repository contains algorithm and data structure solutions implemented in Java. The primary focus is on LeetCode practice problems, with a small Codeforces example included.

## Contents

- `LeetCode/`
  - Java solutions for a range of LeetCode problems.
  - Problems cover topics such as arrays, strings, binary search, backtracking, linked lists, and more.

- `.cph/`
  - A small Codeforces-related folder with one Java solution and problem-related files.

## Highlights

- Clean, individual Java files for each problem.
- Practical practice set for common interview and contest topics.
- Simple structure for easy navigation.

## How to use

1. Open the repository in your Java IDE or editor.
2. Browse `LeetCode/` for problem solutions.
3. Run the selected `.java` file directly if your environment supports it.
4. Use the file names to identify the problem by number and title.

## Example files in `LeetCode/`

- `1. Two Sum.java`
- `3. Longest Substring Without Repeating Characters.java`
- `15.3sum.java`
- `51.N-queens.java`
- `876.middle of linked list.java`

## Notes

- File names are mostly based on problem numbers and titles.
- This repository is intended as a personal practice collection rather than a polished library.
